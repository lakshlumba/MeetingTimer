Meeting Agenda Timer

App will be used to help make sure a meeting involving several talks stays on schedule.

Following are the functionality:

  1. User can schedule maximum of 8 meetings
  2. User can enter the Title, Description, Date and Time of eack talk
  3. User can see list of talks in tabular form
  4. Proper timer is provided for each talk

Color display according to the timer

  1. White color will be displayed, if more than 30% time remaining
  2. Green color will be displayed, if less than 30% time remaining
  3. Yellow color will be displayed, if less than 15% time remaining
  4. Red color will be displayed, if less than 5% time remaining

Technology used:

  1. HTML,CSS,JQuery,JavaScript
